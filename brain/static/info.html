Why am I receiving HTTP requests from this machine?

These HTTP requests are part of an Internet-wide research study being conducted by IT security scientists at the Ruhr University Bochum, Germany. This research involves making benign HTTP requests to every domain in the Tranco top sites ranking. By measuring the entire list of the most popular 1 million domains, we are able to study global trends in web security.

We will never attempt to exploit security problems, guess passwords, or change device configurations. We only receive data that is publicly visible to anyone who visits your website.

We are using the Playwright automation framework to facilitate some of these scans. Playwright has been used for dozens of peer-reviewed academic papers.

Can I request that my website is excluded?

To have your host or network excluded from future scans conducted by us, please contact webscannerinfo@gmail.com and include the domain name you would like us to blacklist. Alternatively, you can configure your firewall to drop traffic from the IP addresses we use for scanning: 134.147.198.123-125.
